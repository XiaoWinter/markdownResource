<h1 align='center'>VS快捷键</h1>



```
alt + shift 竖向一条
ctrl + o 打开项目所在目录
ctrl + p    Go to File
Ctrl+G    Go to Line..
ctrl + shift + [ 收起代码块
ctrl + shift + ] 展开代码块
Ctrl+U   取消上个光标操作
Ctrl+L    选中当前行
Ctrl+Shift+L    选中所有当前词
Shift+Alt+→    扩展选择
Shift+Alt+←    缩小选择
Ctrl+,      用户设置
Alt+↑/ ↓     上下移动行
Shift+Alt+↓/↑    上下复制行
Ctrl+Shift+K     删除行
Ctrl+Enter         向下插行
Ctrl+Shift+Enter       向上插行
Ctrl+Shift+\     Jump to matching bracket
Ctrl+]/ [          Indent/outdent line
Home/ End         行最前/行最后
Ctrl+Home        文件最前
Ctrl+End          文件最后
Ctrl+↑/ ↓        向前一行/向后一行
Alt+PgUp/ PgDn      Scroll page up/down
!+tab 快速生成h5模板网页
```
