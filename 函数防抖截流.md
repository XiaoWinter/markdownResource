<h1 align='center'>函数防抖截流</h1>
#### 函数防抖：

```
在一定时间内频繁触发某事件时，只会执行最后一次触发的事件的事件处理程序
```

```javascript
handleChange=(BraftEditor)=>{
    // console.log(BraftEditor.toHTML());
    let timeId

    if(timeId){
        clearTimeout(timeId)
    }

    timeId = setTimeout(()=>{
        this.setState({
            editorState: BraftEditor
        })
    },3000)

  }
  
```



#### 函数节流：

```
在一定时间内频繁触发某事件时，只会执行第一次触发的事件的事件处理程序
```
