#### 规则6

#### 规则7

从相关的关联中，设法架构处一本书的基本**论述**。

找出关键句

挑出什么是重要的

总重要的句子就是在整个论述中，阐述作者判断的部分。他沟通的主要核心是他说下的肯定与否定的判断，以及他为什么会这么做的理由。

被困惑，而且知道自己被困惑。

##### 找出主旨

主旨一定在一本书最重要的论述中不是前提就是结论，你可以按照书中的论述顺序找到关键的句子。

发现句子表达的主旨，主旨表达的意义更多，需要借助前后关联理解；句子可能不只一个主旨

主旨并非是空泛的，如果你理解了主旨，就可以用其解释或指导实践一些东西，否则，就只是仅仅在背书了。

口语主义：使用文字，没有体会其中传达的思想，或没有注意到其中的意指

##### 找出论述

作者写作的时候，可能分段上并不注意

论述：一系列，先后有序，其中某些还带有提出例证与理由作用的主旨。

在一本书中都有许多段落没有任何论述。

如果可以，找出书中说明重要论述的段落。但是，如果这个论述并没有表达出来，你就要自己去架构（总结表述）出来。你要从这一段或那一段中挑出句子来，然后整理出前后顺序的主旨，以及其组成的论述。

好的论说性书籍的作者会想要说出自己的想法。划分好段落

首先，记住声明

其次，归纳论述（一个或几个事实），演绎论述（通则）

### 如何阅读实用型的书

实用型的书可以提供行事的指导，但是解决实际的问题需要读者的理解运用。

#### 判断是否为一本实用型的书

一本书里包含了规则，原理，准则或是一般性的指导，就可以被认为是实用型的书。

#### 两种类型的实用型书

理论性的原则会归纳出出色的行事规则。通过观察书的侧重点，实用型的书可以分为两种：

1.侧重说明规则的，例如，烹饪书，驾驶指南

2.侧重说明规则的形成原理，例如，经济，政治类的书

可能某些实用型的书这两种侧重的偏向并不明显



##### 阅读侧重规则的书

找出规则句

##### 阅读侧重原理的书

找出原理，从原理中找到可以理解的规则，正确的行为规则应该是：1.真的有效，2.可以让你走向你的期望

##### 评论一本实用型的书

一本实用型的书的评价，核心理念是：所有的事都与结果和目标有关。

##### 找主旨或论述的实用规则

1.作者的目的，或者是要解决什么问题

2.作者建议用什么方法到达这个目的，或怎么解决问题

##### 读整本书的要想的问题

1.一定要知道作者想要解决什么？（非常重要）

2.作者细说了什么？作者想让我怎么做？为什么让我那么做？

3.内容正确吗？

4.这本书与我何干？

## 分析阅读规则

第一阶段,找出**一本书在谈什么**的规则

```
1.依照书本的种类与主题做分类。
2.用最简短的句子说出整本书在谈什么。
3.按照顺序关系，列出全书的重要部分。将全书纲要拟出来之后，再将各个部分的纲要也一一列出。
4.找出作者在问的问题，或作者想解决的问题。
```

第二阶段，**诠释一本书的内容**的规则

```
1.诠释作者使用的关键字，与作者达成共识。
2.从最重要的句子中抓出作者的重要主旨。
3.找出作者的论述，重新架构这些论述的前因后果，以明白作者的主张。
4.确定作者已经解决了那些问题，还有哪些是未解决的。在未解决的问题中，确认哪些是作者认为自己无法解决的问题。
```

第三阶段：像是沟通知识一样**评论一本书**的规则

```
智慧的礼节
1.除非你已经完成大纲架构，也能诠释整本书了，否则不要轻易批评。
2.不要争强好胜，非要辩到底不可。
3.在说出评论之前，你要能证明自己区别得出真正的知识与个人观点的不同。
批评观点的特别标准
1.证明作者知识不足。
2.证明作者的知识错误。
3.证明作者不合逻辑。
4.证明作者的分析与理由是不完整的。（对本书暂缓评论）
```
